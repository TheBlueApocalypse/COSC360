Code Explanation:

Scatter Plots:

The code linked below will print out the the scatter plots for height vs temperature, height vs rain, height vs altitude, and height vs location in that order. The scatter plot is a way of showing the distribution of data for a non linear set of data.

Calculation:

Then the code makes comparisons and calculations on the line of best fit. A line of best fit is a line that runs trough a scatter plot . Once the line equation is created the data is then used calculate the which line has the best data set. The calculations include solvinhg the function YMean = m(XMean)+b.

Comparison:

The best data set means which site of data is representitive of the given line or which group sits closest to the line of best fit. Once R^2, the value that shows the accuracy of the line of best fit, is found for each group the code compares each value and then prints out the data that influences height of trees the most.
